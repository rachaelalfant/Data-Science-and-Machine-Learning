# Data Science & Machine Learning

This is a repository containing various implementations of algorithms introduced in INDE 577: Data Science and Machine Learning at Rice University. The course introduces fundamentals of data science and machine learning, with topics covered including, but not limited to: Data Visualization, Linear and Logistic Regression, K-nearest Neighbors, K-Means Clustering, Neural Networks, the Perceptron/Multilayer Perceptron learning algorithms, Decision Trees, and Ensemble Learning/Random Forest Classifiers. All of the files in this repository are Jupyter notebooks containing projects in Python v.3.
